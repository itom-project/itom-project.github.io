ITOM Software¶

ITOM is an open source software suite for operating measurement systems, laboratory automation and data evaluation.

Main applications of itom:
  • Development and operation of sensors and measurement systems
  • Communicate with wide range of hardware systems (cameras, actuators, …)
  • Diversified set of evaluation and data processing methods
  • Rapid prototyping of measurement and inspection setups
  • Change system parameters or components easily at runtime
  • Embedded scripting language
  • Extend the graphical user interface GUI by system adapted dialogs and windows
