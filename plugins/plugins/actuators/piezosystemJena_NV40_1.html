2.12. Piezosystem Jena NV40/1 CLE¶

Summary: Piezosystem Jena NV40/1 CLE
Type: Actuator
License: LGPL
Platforms: Windows, Linux
Devices: One axis piezo actuator from Piezosystem Jena (NV 40/1 CLE)
Author: V. Ferreras-Paz, M. Gronle, W. Lyda, ITO, University Stuttgart
Version: 0.1.0

2.12.1. Overview¶

The PiezosystemJena is an itom-plugin, which can be used to communicate with the one-axis piezo controller Piezosystem Jena NV40/1 CLE. This system needs a serial port. The parameters are set automatically during initialization. It is initialized by actuator(“PiezosystemJena_NV40_1”, serialInstance).

2.12.2. Initialization¶

The following parameters are mandatory or optional for initializing an instance of this plugin:

  • serial: {dataIO|actuator}
    An opened serial port (the right communcation parameters will be set by this piezo-controller).
