loadRawImage(filename, image[, arguments, usetmpdir])¶

import filter for (dslr) raw images based on dcraw

Parameters:
  • filename (str) – file to open
  • image (dataObject) – Empty object, will contain 2D image later
  • arguments (str - optional) – commandline arguments for dcraw, default is -D -j -t0, i.e. raw data no scaling
  • usetmpdir (int - optional) – do temporal unpacking in temporary directory
