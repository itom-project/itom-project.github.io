OpenCV Filters¶

Summary: Wrapped algorithms from OpenCV
Type: Algorithm
License: LGPL
Platforms: Windows, Linux
Author: W. Lyda, M. Gronle, ITO, University Stuttgart

Overview¶

This plugin provides wrappers for various OpenCV algorithms. These are for instance:

  • morphological filters (dilation, erosion)
  • image filtering (blur, median blur...)
  • 1d and 2d fft and ifft
  • histogram determination
  • feature detections (circles, chessboard corners...)

This plugin not only requires access to the core library of OpenCV but also to further libraries like imgproc and calib3d.

These filters are defined in the plugin:

  1. cvFFT2D
  2. cvBlur
  3. cvRotateM90
  4. cvRemoveSpikes
  5. cvFlipLeftRight
  6. cvRemapParams
  7. cvIFFT2D
  8. cvRotateP90
  9. cvMergeChannels
  10. cvFindFundamentalMat
  11. cvFlannBasedMatcher
  12. cvDrawChessboardCorners
  13. cvFindChessboardCorners
  14. cvMedianBlur
  15. cvDrawMatcher
  16. cvUndistort
  17. cvFindCircles
  18. cvUndistortPoints
  19. cvFFT1D
  20. cvComputeCorrespondEpilines
  21. cvDrawKeypoints
  22. cvEstimateAffine3DParams
  23. cvSplitChannels
  24. cvCalibrateCamera
  25. cvIFFT1D
  26. cvErode
  27. cvRotate180
  28. cvResize
  29. cvCornerSubPix
  30. cvFindHomography
  31. cvDilate
  32. cvFlipUpDown
  33. cvInitUndistortRectifyMap
