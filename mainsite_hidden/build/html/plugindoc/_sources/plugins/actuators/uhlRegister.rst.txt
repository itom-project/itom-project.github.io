Parameters
===========

**accel**: {float}
    
**async**: {int}
    
**comPort**: {int}, read-only
    The current com-port ID of this specific device. -1 means undefined

**inversex**: {int}
    
**inversey**: {int}
    
**inversez**: {int}
    
**joyenabled**: {int}
    Enabled/disabled Joystick. Default: enabled

**name**: {str}, read-only
    
**numaxis**: {int}, read-only
    
**speed**: {float}
    
**timeout**: {float}
    timeout for axes movements in seconds
