OpenCV Filters¶

Summary: Wrapped algorithms from OpenCV
Type: Algorithm
License: LGPL
Platforms: Windows, Linux
Author: W. Lyda, M. Gronle, ITO, University Stuttgart

Overview¶

This plugin provides wrappers for various OpenCV algorithms. These are for instance:

  • morphological filters (dilation, erosion)
  • image filtering (blur, median blur...)
  • 1d and 2d fft and ifft
  • histogram determination
  • feature detections (circles, chessboard corners...)

This plugin not only requires access to the core library of OpenCV but also to further libraries like imgproc and calib3d.

These filters are defined in the plugin:

  1. cvRotateP90
  2. cvBlur
  3. cvRemapParams
  4. cvFindCircles
  5. cvMedianBlur
  6. cvInitUndistortRectifyMap
  7. cvResize
  8. cvUndistort
  9. cvFFT2D
  10. cvDrawMatcher
  11. cvFlipLeftRight
  12. cvCalibrateCamera
  13. cvRotate180
  14. cvComputeCorrespondEpilines
  15. cvDrawKeypoints
  16. cvIFFT1D
  17. cvFindHomography
  18. cvDilate
  19. cvFlannBasedMatcher
  20. cvMergeChannels
  21. cvIFFT2D
  22. cvRemoveSpikes
  23. cvFlipUpDown
  24. cvRotateM90
  25. cvEstimateAffine3DParams
  26. cvFindFundamentalMat
  27. cvUndistortPoints
  28. cvFFT1D
  29. cvFindChessboardCorners
  30. cvSplitChannels
  31. cvErode
