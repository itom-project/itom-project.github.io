Basic Filters¶

Summary: ITO developed filter-functions for dateObjects
Type: Algorithm
License: LGPL
Platforms: Windows, Linux
Author: W. Lyda, T. Boettcher, University Stuttgart

Overview¶

This plugin provides several basic filter calculations for itom::dataObject. These are for instance:

  • merging of planes
  • swap byte order of objects
  • resample slices from dataObjects
  • mean value filter along axis

This plugin does not have any unusual dependencies.

These filters are defined in the plugin:

  1. swapByteOrder
  2. clipValues
  3. calcHist
  4. calcObjSlice
  5. highValueFilter
  6. fillObject
  7. flatten3Dto2D
  8. calcMeanZ
  9. mergeColorPlane
  10. replaceInfAndNaN
  11. lowPassFilter
  12. spikeMedianFilter
  13. clipAbyB
  14. medianFilter
  15. gaussianFilterEpsilon
  16. lowValueFilter
  17. gaussianFilter
