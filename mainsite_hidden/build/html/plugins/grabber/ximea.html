Ximea¶

Summary: Ximea xiQ-Camera
Type: DataIO
License: LGPL / do not copy Ximea-DLLs
Platforms: Windows, (Linux possible but yet not implemented)
Devices: Cameras from company Ximea
Author: C. Kohler, twip optical solutions GmbH, Stuttgart

Overview¶

Developed for Windows only. Tested with Ximea xIQ-cameras for black and white.

Initialization¶

The following parameters are mandatory or optional for initializing an instance of this plugin:

  • camera Number: {int (int), optional} [0,3], default: 0, step: 1

    The index of the addressed camera starting with 0

  • restoreLast: {int (int), optional} [0,1], default: 0, step: 1

    Toogle if the driver should try to connect to the last initilized camera

  • bandwidthLimit: {int (int), optional} [0,100000], default: 0, step: 1

    bandwidth limit in Mb/sec. If 0 the maximum bandwidth of the USB3 controller is used [default]. The allowed value range depends on the device and will be checked at startup.

Changelog¶

  • itom setup 1.2.0: This plugin has been compiled using the XimeaAPI 4.0.0.5
