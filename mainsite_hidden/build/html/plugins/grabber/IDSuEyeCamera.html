IDS uEye¶

Summary: IDS uEye grabber.
Type: DataIO
License: Licensed under LGPL
Platforms: Windows, Linux ready but not tested
Devices: IDS Imaging cameras
Author: M. Gronle (ITO Uni Stuttgart); A. Kempe (ILT Aachen, Pulsar Photonics)

Overview¶

This plugin supports IDS uEye cameras and has currently been tested with the following models: - UI145xSE-C (colored, USB2) - UI124xSE-M (monochrome, USB2). - UI337xCP-C (colored, USB3)

The plugin has been compiled using the IDS library version 4.40.

In order to run your camera, please install the SDK imaging software in the right version such that the necessary drivers are installed.

The first draft of this plugin has been implemented by Pulsar Photonics GmbH; further work has been done by ITO, University of Stuttgart.

Initialization¶

The following parameters are mandatory or optional for initializing an instance of this plugin:

  • camera_id: {int (int), optional} [0,254], default: 0, step: 1

    Camera ID of the camera to open (0: the next free camera will opened [default], 1-254: else)

  • color_mode: {str (char*), optional} default: auto

    initial color model of camera (‘gray’, ‘color’ or ‘auto’ (default)). ‘color’ is only possible for color cameras

  • debug_mode: {int (int), optional} [0,1], default: 0, step: 1

    If debug_mode is 1, message boxes from the uEye driver will appear in case of an error (default: off, 0)
