1. delete build folder
2. all plugin have to be compiled on your PC and be loaded by itom
3. necessary python packages: Sphinx, sphinx-fulltoc, sphinx-rtd-theme
4. start update_plugindoc_source_plugins.py in itom to update the plugin documentation from the source rst-files
5. start create_plugin_documentation.py to create plugin docu for the website
6. start create_html_documentation.py to create the itom website
