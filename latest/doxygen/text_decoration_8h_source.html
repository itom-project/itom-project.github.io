60 /*
61 Helper class to quickly create a text decoration. The text decoration is an
62 utility class that adds a few utility methods to QTextEdit.ExtraSelection.
63 
64 In addition to the helper methods, a tooltip can be added to a decoration.
65 (useful for errors markers and so on...)
66 
67 Text decoration expose a **clicked** signal stored in a separate QObject:
68  :attr:`pyqode.core.api.TextDecoration.Signals`
69 
70 .. code-block:: python
71  deco = TextDecoration()
72  deco.signals.clicked.connect(a_slot)
73  def a_slot(decoration):
74  print(decoration)
75 */

77 {
